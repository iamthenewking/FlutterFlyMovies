# FlutterFlyMovies

A Movies API(TMDB) demo Flutter project.

This app demonstrates the following customization in Flutter

- Animation between pages
- Animating the objects
- Fetching JSON from API
- Customizing Fonts
- Customizing Tabs
- using widgets to layout the UI 
- Handling the states
